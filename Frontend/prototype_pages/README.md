# MindEase - 原型页面

##  项目概述

MindEase 用户界面原型系统。融入现代、自然的视觉设计。
**！！！！！注意**：
本设计为原型展示，不具备实际功能，仅作为外观设计目标

## 文件结构

```
prototype_pages/
├── index_4.html              # 主导航页面（侧边栏+iframe）
├── home_4.html               # 用户首页（网格布局）
├── mood-diary_4.html         # 情绪日记编辑页

├── emotion-report_4.html     # 用户情绪报告页
├── shared-styles-4.css       # 共享样式配置
└── README.md                 # 本文档
```

## 快速开始

### 方式一：直接浏览
1. 打开 `index_4.html` 作为主入口
2. 通过左侧侧边栏（hover展开）切换不同页面
3. 页面通过 iframe 加载

### 方式二：本地服务器
```bash
# 在文件夹中运行
python -m http.server 8080

# 然后访问
http://localhost:8080/index_4.html
```

## 核心设计

### 1. **鼠尾草绿配色方案**
- **主色调**：鼠尾草绿 (#7B9E89) - 平静、自然
- **背景色**：浅绿灰 (#F2F5F3) - 温和、舒适
- **深色文字**：深岩灰 (#2C3E50) - 专业、清晰
- **暖沙色**：#E8E1D9 - 温暖、自然
- **陶土色强调**：#C47C6B - 活力、重点

### 2. **高级质感纹理**
- SVG 噪点纹理叠加层（4% 透明度）
- 浮动彩色圆球（orbs）- 紫色、绿色、蓝色
- 持续的呼吸式动画（10s 循环）

### 3. **增强型玻璃态设计**
- 60-70% 白色透明度
- 24px backdrop blur
- 精致的白色边框和阴影
- 悬停时的微交互动画

### 4. **优雅的侧边栏**
- Hover 展开动画（24px → 256px）
- cubic-bezier 缓动函数
- 文字淡入效果with延迟
- 选中状态白色背景+缩放

### 5. **精致的交互动画**
- Float 浮动效果（情绪表情）
- Fade-in 淡入动画
- Slide-up 上滑动画
- 卡片悬停上浮

## vue3下部分设计规范

### 主题色
- 主色：`#7b9e89` (鼠尾草绿)
- 深色：`#5f7a6a`
- 背景：`#f2f5f3`
- 暖色：`#c47c6b`

### 组件使用
优先使用 Element Plus 组件：
- 表单：`el-form`, `el-input`, `el-button`
- 布局：`el-row`, `el-col`, `el-card`
- 导航：`el-menu`, `el-dropdown`
- 反馈：`ElMessage`, `ElNotification`
