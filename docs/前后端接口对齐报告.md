## 💡 联调技巧

### 查看请求/响应

1. 打开浏览器开发者工具（F12）
2. Network → XHR
3. 检查请求 URL、请求头、响应数据

### 常见错误排查

| 错误      | 原因            | 解决方案                     |
| --------- | --------------- | ---------------------------- |
| CORS 错误 | 跨域配置问题    | 已修复（WebCorsConfig.java） |
| 404 错误  | 接口路径不存在  | 检查接口是否已实现           |
| 500 错误  | 后端逻辑错误    | 查看后端控制台日志           |
| 401 错误  | Token 无效/过期 | 重新登录获取新 Token         |
